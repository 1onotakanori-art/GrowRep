## 🎯 来週のチャレンジ（確定カード）
（**`nextWeekConfirmed` があればそれを主役に**。日曜17時の抽選で来週の3種目はもう確定している＝予想ではなく「確定カード解説」。`nextWeekConfirmed` が null のとき＝まだ選出前のときだけ `nextWeekForecast` の確率予想にフォールバックする）

`nextWeekConfirmed.exercises` がある場合:
- **来週の3種目を発表**: 各種目名（バーバリアンは「短いほど良い」と明記）を「来週はこの3つで決まり！」と断言して紹介。
- **出たら本命**: 各種目の `favorite`（全期間ベスト記録保持者）と `contenders`（上位3人）で「この種目は○○が本命、△△が肉薄」と当日のカードを実況。
- **得意不得意を踏まえた展開**: 確定3種目と各メンバーの `weeklyChallenge` 順位・`strengths`/`tagStrength` を結びつけ「○○有利／□□は巻き返しの好機」を描く。月間ダービーの順位（`monthlyDerby`）も絡めて「逆転のチャンス週になるか」。

`nextWeekConfirmed` が null（選出前）の場合のみ — `nextWeekForecast` で**確率予想**（未確定と必ず断る）:
- バーバリアン枠の最有力（`barbarianCandidates` の `pickChance` 上位）＋ `favorite`。
- 通常枠はほぼ均等抽選で大荒れ、と煽る。
