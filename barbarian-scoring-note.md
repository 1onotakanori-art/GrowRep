# 前提の再整理

| 条件 | 内容 |
|---|---|
| 通常種目 | 1位が100点（100%）、他の人は1位との比率で得点 |
| タイムアタック | 短い秒数ほど高得点 |
| 面白さの核 | 1位の記録次第で全員の得点が変動する |
| 得点上限 | 1位 = 100点（100%） |

ポイントは、**「絶対的な基準時間」ではなく「その週の1位のタイム」が基準になる**ということです。これにより：

- 誰かが驚異的な記録を出す → 全員の得点が下がる
- 1位が平凡な記録 → 自分の得点が相対的に上がる

この「1位次第で変わる」仕組みは、通常種目（回数系）と同じ構造にできます。

---

# 提案: 1位タイム基準の比率スコア

## 通常種目（回数系）の既存ロジック

\[
\text{score} = \frac{\text{自分の回数}}{\text{1位の回数}} \times 100
\]

→ 多いほど高い。1位は100点。

## タイムアタック（秒数系）の対応ロジック

\[
\text{score} = \frac{T_{\text{best}}}{T_{\text{self}}} \times 100
\]

- \(T_{\text{best}}\): その週の1位（最短）タイム
- \(T_{\text{self}}\): 自分のタイム

### 例: 1位が30秒の場合

| 自分のタイム | 計算 | 得点 |
|---|---|---|
| 30秒（1位） | 30/30 × 100 | 100点 |
| 45秒 | 30/45 × 100 | 66.7点 |
| 60秒 | 30/60 × 100 | 50.0点 |
| 90秒 | 30/90 × 100 | 33.3点 |
| 120秒 | 30/120 × 100 | 25.0点 |

---

## 「1位次第で変わる面白さ」の確認

同じ60秒の記録でも：

| 1位のタイム | 自分（60秒）の得点 |
|---|---|
| 55秒 | 91.7点 |
| 40秒 | 66.7点 |
| 30秒 | 50.0点 |
| 20秒 | 33.3点 |

→ 1位が更新されるたびに自分の得点がリアルタイムで変動する。通常種目と同じ面白さが保たれます。
